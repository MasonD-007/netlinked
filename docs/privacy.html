Privacy Policy

Last updated: April 02, 2025

This Privacy Policy describes how NetLinked ("We", "Us", or "Our") handles your information when you use our Chrome extension for generating personalized LinkedIn connection messages using AI. We take your privacy seriously and want to be transparent about how we collect, use, and protect your data.

What NetLinked Does

NetLinked is a Chrome extension that serves one specific purpose: helping users create personalized connection messages on LinkedIn using AI. It leverages the Gemini API to analyze LinkedIn profiles and generate relevant connection requests that stand out.

Data Collection and Processing

When you use NetLinked, we process the following types of information:

  • LinkedIn Profile Data: The extension automatically extracts publicly visible information from LinkedIn profiles you view. This data is only processed locally on your machine.
  • Gemini API Usage: When generating messages, the extension communicates with the Gemini API using your provided API key.
  • Extension Settings: Your preferences and API key are stored securely in Chrome's extension storage.
  • Feedback Data: If you choose to provide feedback, it is sent directly to the project maintainer to improve the service.

How We Use Your Data

Your data is used exclusively for:

  • Generating personalized connection messages
  • Improving the extension's functionality through your feedback
  • Storing your preferences and settings

Data Security

We prioritize the security of your data through the following measures:

  • All data processing occurs locally on your machine
  • API keys are stored securely in Chrome's extension storage
  • No personal information is collected or stored on our servers
  • Communication with the Gemini API is encrypted
