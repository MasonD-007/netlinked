
1. profile-card
    - name
    - headline
    - location
    - Amount of connections
2. Highlights
    - anything
3. About
    - Description
4. Activity (open more tab) "(profile-url)/recent-activity/all/"
    - Posts (only for rn)
    - Comments
5. Experience (open more tab) "(profile-url)/details/experience/"
    - List of all experiences
    - Each experience has:
        - Title
        - Company
        - Location
        - Start date
        - End date
        - Description
6. Education (open more tab) "(profile-url)/details/education/"
    - List of all education
    - Each education has:
        - School name
        - Degree
        - Field of study
        - Start date
        - End date
        - Activites and societies
7. Licenses & Certifications (open more tab) "(profile-url)/details/certifications/"
    - List of all licenses and certifications
    - Each license or certification has:
        - License or certification name
        - Issuing organization
        - Date of issue
8. Recommendations (open more tab) "(profile-url)/details/recommendations/"
    - List of all recommendations
    - Each recommendation has:
        - Name of recommender
        - Position of recommender
        - Recommendation text
9. Projects (open more tab) "(profile-url)/details/projects/"
    - List of all projects
    - Each project has:
        - Title
        - Description
        - Technologies used
10. Honors & Awards (open more tab) "(profile-url)/details/honors/"
    - List of all honors and awards
    - Each honor or award has:
        - Title
        - Issuing organization
        - Date of issue
11. Skills (open more tab) "(profile-url)/details/skills/"
    - List of all skills
    - Each skill has:
        - Skill name
        - Skill proficiency level
